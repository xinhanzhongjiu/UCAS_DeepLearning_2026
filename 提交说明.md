# 实验7 提交说明

## 文件结构

- `实验7_神经网络语言模型实验报告.docx`：最终实验报告。
- `实验7_神经网络语言模型汇报PPT.pptx`：讲解幻灯片。
- `code/train_ptb_lstm.py`：实验代码。
- `code/requirements.txt`：Python 依赖。
- `code/README.md`：代码运行说明。
- `checkpoints/ptb_lstm.pt`：基础 LSTM 最优模型权重。
- `checkpoints/ptb_lstm_locked_sgd.pt`：locked dropout 微调模型权重。
- `docs/`：实验指导书和作业要求。

## 复现最终结果

```powershell
python -u train_ptb_lstm.py --preset medium --ensemble-eval checkpoints/ptb_lstm.pt checkpoints/ptb_lstm_locked_sgd.pt --batch-size 1 --cache-size 750 --cache-lambda 0.11 --cache-theta 0.52 --cuda
```

最终结果：

```text
Cached ensemble valid ppl: 69.964
Cached ensemble test ppl: 69.018
```
